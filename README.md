# Java Payroll System (v1)

This project is a simple employee payroll calculator developed using Java. It calculates the total income based on the employee's category (Grade A, B, or C) and overtime hours worked.

## Key Features
- **Dynamic Salary Calculation**: Determines base salary based on employee grade.
- **Overtime Logic**: Calculates bonuses using percentage-based logic for extra work hours.
- **Interactive Console**: Built using Java Scanner for user input.

## Implementation Details
- Language: Java
- Tools: Scanner, If-Else Statements, Arithmetic Operations
- Context: University Project / Portfolio Piece
